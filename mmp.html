About MMP

MMP — this is a selected subset of standard metadata for supplying of the basic needs of the life cycle of photos, namely:

  • unique naming (in accordance with NOP);
  • storing of information about date and time of shooting;
  • storing of information about authorship;
  • storing of information about copyright;
  • storing of information about owner of the deviceused to create the photo;
  • storing of description of the visual component of the photo.
  • storing of necessary additional information.

MMP is presented as a combined list of tags from several relevant metadata standards, which is created as a single source of these tags specifically for convenience.

Tags for MMP are taken from the following standards:

  • CIPA DC-010-2020 (Exif 2.32 metadata for XMP)
  • IPTC Photo Metadata Standard 2021.1 (IPTC Core schema 1.3)
  • IPTC Photo Metadata Standard 2021.1 (IPTC Extension schema 1.6)
  • PLUS Version 1.2.3
  • XMP (XMP SPECIFICATION PART 1. DATA MODEL, SERIALIZATION, AND CORE PROPERTIES. April 2012)
  • XMP (XMP SPECIFICATION PART 2. ADDITIONAL PROPERTIES. August 2016)

Notes are taken from the standards and presented in its original.

Additionally each standard tag of metadata is mapped to ExifTool-tag.

Symbol => is indicated ExifTool-tags, which are differ at the basis of its name from the names of standard tags

When using ExifTool to add metadata structures, it is enough to add only child fields, ExifTool creates and adds parent fields automatically.
