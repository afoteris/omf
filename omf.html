Об ОМФ

ОМФ — это отобранное подмножество стандартных метаданных для снабжения основных потребностей жизненного цикла фотографий, а именно:

  • уникальное именование (в соответствии с ПИФ);
  • хранение информации о дате и времени съёмки;
  • хранение информации об авторстве;
  • хранение информации об авторских правах;
  • хранение информации о владельце устройства, с помощью которого была создана фотография;
  • хранение описания визуальной составляющей фотографии.
  • хранение необходимой дополнительной информации.

ОМФ представлены в виде объединённого списка тегов (Объединённый список тегов) из нескольких соответствующих стандартов метаданных (Постандартные списки тегов), который создан специально для удобства как единый источник этих тегов.

Теги для ОМФ взяты из следующих стандартов:

  • CIPA DC-010-2020 (Exif 2.32 metadata for XMP)
  • IPTC Photo Metadata Standard 2021.1 (IPTC Core schema 1.3)
  • IPTC Photo Metadata Standard 2021.1 (IPTC Extension schema 1.6)
  • PLUS Version 1.2.3
  • XMP (XMP SPECIFICATION PART 1. DATA MODEL, SERIALIZATION, AND CORE PROPERTIES. April 2012)
  • XMP (XMP SPECIFICATION PART 2. ADDITIONAL PROPERTIES. August 2016)

Примечания взяты из стандартов и представлены преимущественно в оригинале.

Дополнительно каждому стандартному тегу метаданных сопоставлен ExifTool-тег.

Знаком => указаны ExifTool-теги, которые в основе своего названия отличаются от названий стандартных тегов.

При использовании ExifTool для добавления структур метаданных, достаточно добавлять только дочерние поля, родительские поля ExifTool создаёт и добавляет автоматически.

Постандартные списки тегов

Эти списки составлены индивидуально по каждому из представленных стандартов, не учитывая возможные повторения из-за пересечения списков между собой.

CIPA DC-010-2020

Exif 2.32 metadata for XMP

Родной             | XMP-аналог              | Примечание
=================================================================================================================================================================================================================================================================================
ImageDescription   | dc:description          | The title of the image as an ASCII string.
Model              | tiff:Model              | Model name or number of the equipment as an ASCII string.
Artist             | dc:creator              | Camera owner, photographer or image creator as an ASCII string.
Copyright          | dc:rights               | Copyright information as an ASCII string.
UserComment        | exif:UserComment        | Comments from user.
DateTimeOriginal   | exif:DateTimeOriginal   | Date and time when the original data was generated (no time zone in Exif), stored in ISO 8601 format, not in the original Exif format. This property includes the value for the Exif SubSecTimeOriginal (37521, 0x9291) attribute.
OffsetTimeOriginal |                         | Offset from UTC of the time of DateTimeOriginal.
FileSource         | exif:FileSource         | Indicates image source:
                                             | 0 = others
                                             | 1 = scanner of transparent type
                                             | 2 = scanner of reflex type
                                             | 3 = DSC
CameraOwnerName    | exifEX:CameraOwnerName  | This tag records the owner of a camera used in photography as an ASCII string.
BodySerialNumber   | exifEX:BodySerialNumber | The serial number of the camera or camera body used to take the photograph.

IPTC Photo Metadata Standard 2021.1

IPTC Core schema 1.3

Родной             | XMP-аналог           | Примечание
==============================================================================================================================================================================================================================================
Copyright Notice   | dc:rights            | Enter a Notice on the current owner of the Copyright for this image, such as ©2008 Jane Doe.
Creator            | dc:creator           | Enter the name of the person that created this image.
Credit Line        | photoshop:Credit     | Enter who should be credited when this image is published.
Description        | dc:description       | Enter a "caption" describing the who, what, and why of what is happening in this image, this might include names of people, and/or their role in the action that is taking place within the image.
Keywords           | dc:subject           | Enter any number of keywords, terms or phrases used to express the subject matter in the image.
Rights Usage Terms | xmpRights:UsageTerms | Enter instructions on how this image can legally be used.
Title              | dc:title             | Enter a short verbal and human readable name for the image, this may be the file name.
