SSD for Windows 10

1. выделить место 10%
2. отключить индексацию
3. отключить дефрагментацию
   Task Sheduler -> … -> Microsoft -> Windows -> Defrag
5. отключить гибернацию (по желанию, но она нужна для быстрой загрузки)
   powercfg -h off
   https://docs.microsoft.com/en-us/troubleshoot/windows-client/deployment/disable-and-re-enable-hibernation
4. отключить быструю загрузку системы
   https://www.asus.com/ru/support/FAQ/1031533
